# Sorting_Algos_Visualizer
Web app built to visualize some of the popular sorting algorithms for better understanding. Algorithms which are currently implemented are:<br>
* Bubble sort
* Merge sort
* Insertion sort
* Selection sort
# Languages used:
  * Javascript
  * HTML
  * CSS
